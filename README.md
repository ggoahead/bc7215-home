# bc7215-home

ESPHome based air conditioner controller using:

- ESP8266
- BC7215A IR controller
- DHT22
- Home Assistant API


## Architecture

ESP8266
|
├── ESPHome
├── BC7215A Component
├── DHT22
├── Home Assistant API
└── OTA
